# Dental AI Visibility

A lead-generation and audit project for helping dentists, orthodontists, and cosmetic dental clinics understand whether AI search tools can find, understand, and recommend their clinic.

## Core offer

**Free Dental AI Visibility Audit**

> When patients ask ChatGPT for the best dentist near them, does your clinic show up?

## Current assets

- `docs/ai-seo/tcare-dental-sample-ai-visibility-audit.md` — sample audit using TCare Dental Centre
- `docs/ai-seo/dental-ai-visibility-funnel.md` — full funnel, offer ladder, emails, outreach
- `docs/ai-seo/dental-ai-visibility-landing-page-copy.md` — page-ready copy for the MVP landing page

## MVP plan

1. Build a landing page for the free dental AI visibility audit.
2. Capture clinic details through a form.
3. Store leads in Supabase.
4. Send confirmation email.
5. Generate/review audit report.
6. Email report and CTA to book strategy call.
